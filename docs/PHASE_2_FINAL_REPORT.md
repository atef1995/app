### Student Journey

2. **Fork & Code** ‚Üí Implement feature in forked repository
3. **Submit PR** ‚Üí Paste PR URL into platform

8. **PR Merge** ‚Üí Code merged to main branch
9. **Celebration** ‚Üí Confetti, XP earned, badge unlocked
10. **Portfolio** ‚Üí PR added to public profile

1. **Review Assignment** ‚Üí Receive notification of new PR to review

3. **Conduct Review** ‚Üí Use rubric to score PR (functionality, code quality, best practices, documentation)
4. **Submit Feedback** ‚Üí Provide strengths, improvements, suggestions
5. **Earn XP** ‚Üí Receive 25 XP for completing peer review

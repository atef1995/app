# CSS Grid Layout: Master 2D Layouts

CSS Grid is the most powerful layout system in CSS. Unlike Flexbox which is one-dimensional, Grid lets you work with both rows and columns simultaneously, making it perfect for complex web layouts.

Ready to become a Grid master? Let's build some amazing layouts!

## Grid Basics: Your First Grid

CSS Grid works with two main components:
- **Grid Container**: The parent element with `display: grid`
- **Grid Items**: The direct children that get placed in the grid

**Grid Line Concepts:**
- Grid lines are numbered starting from 1
- `grid-column: 2 / 4` means "start at line 2, end at line 4"
- You can also use `span`: `grid-column: span 2` (span 2 columns)
- Negative numbers count from the end: `grid-column: -1` (last line)

## Grid vs Flexbox: When to Use What?

Understanding when to use each layout system is crucial:

### Grid is Perfect For:
- âœ… **2D Layouts** - Working with rows AND columns
- âœ… **Complex Layouts** - Dashboards, magazine-style designs
- âœ… **Precise Positioning** - Exact placement of elements
- âœ… **Overlapping Items** - Creative layered designs
- âœ… **Template Areas** - Visual layout definition

### Flexbox is Perfect For:
- âœ… **1D Layouts** - Either rows OR columns
- âœ… **Component Layout** - Navigation bars, button groups
- âœ… **Centering** - Quick alignment solutions
- âœ… **Space Distribution** - Flexible item sizing
- âœ… **Content-Driven** - Size based on content

**Pro Tip:** Use them together! Grid for page layout, Flexbox for components.

## Grid Best Practices

### 1. Mobile-First Approach
Start simple, add complexity:

```css
.responsive-layout {
  display: grid;
  gap: 20px;
  /* Mobile: single column */
}

@media (min-width: 768px) {
  .responsive-layout {
    grid-template-columns: 1fr 2fr;
  }
}

@media (min-width: 1024px) {
  .responsive-layout {
    grid-template-columns: 200px 1fr 200px;
  }
}
```

### 2. Use Semantic Grid Areas
Make your CSS readable:

```css
.page-layout {
  grid-template-areas:
    "header"
    "navigation"
    "main"
    "sidebar"
    "footer";
}
```

### 3. Combine with Modern CSS
```css
.modern-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(min(300px, 100%), 1fr));
  gap: clamp(1rem, 4vw, 2rem);
}
```

### 4. Fallbacks for Older Browsers
```css
.grid-container {
  /* Flexbox fallback */
  display: flex;
  flex-wrap: wrap;
}

@supports (display: grid) {
  .grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  }
}
```

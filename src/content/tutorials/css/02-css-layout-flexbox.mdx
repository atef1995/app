## What is Flexbox?

**Flexbox** (Flexible Box Layout) is a one-dimensional layout method that allows you to arrange items in rows or columns. It provides powerful alignment capabilities and can automatically adjust spacing.

## Flex Direction

Controls the main axis direction:

```css
.container {
  display: flex;
  flex-direction: row;        /* Default: left to right */
  flex-direction: row-reverse; /* Right to left */
  flex-direction: column;     /* Top to bottom */
  flex-direction: column-reverse; /* Bottom to top */
}
```

### Example: Vertical Layout
```css
.vertical-container {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.header { background: #3498db; }
.main { background: #ecf0f1; flex: 1; }
.footer { background: #34495e; }
```

## Align Items (Cross Axis Alignment)

Controls alignment perpendicular to the main axis:

```css
.container {
  display: flex;
  height: 200px; /* Need height to see vertical alignment */
  align-items: stretch;     /* Default: stretch to fill */
  align-items: flex-start;  /* Top of container */
  align-items: flex-end;    /* Bottom of container */
  align-items: center;      /* Center vertically */
  align-items: baseline;    /* Align to text baseline */
}
```

### Perfect Centering
```css
.center-everything {
  display: flex;
  justify-content: center;  /* Horizontal center */
  align-items: center;      /* Vertical center */
  height: 100vh;           /* Full viewport height */
}
```

## Flex Wrap

Controls whether items wrap to new lines:

```css
.container {
  display: flex;
  flex-wrap: nowrap;  /* Default: all items on one line */
  flex-wrap: wrap;    /* Items wrap to new lines */
  flex-wrap: wrap-reverse; /* Items wrap in reverse order */
}

/* Shorthand */
.container {
  display: flex;
  flex-flow: row wrap; /* flex-direction + flex-wrap */
}
```

## Align Content (Multi-line Alignment)

Works when items wrap to multiple lines:

```css
.container {
  display: flex;
  flex-wrap: wrap;
  height: 400px;
  align-content: flex-start;    /* Lines packed to start */
  align-content: flex-end;      /* Lines packed to end */
  align-content: center;        /* Lines centered */
  align-content: space-between; /* Space between lines */
  align-content: space-around;  /* Space around lines */
  align-content: stretch;       /* Lines stretch to fill */
}
```

## Flex Item Properties

### Flex Grow
Controls how much an item should grow:

```css
.item-1 { flex-grow: 1; } /* Takes 1/3 of extra space */
.item-2 { flex-grow: 2; } /* Takes 2/3 of extra space */
.item-3 { flex-grow: 1; } /* Takes 1/3 of extra space */
```

### Flex Shrink
Controls how much an item should shrink:

```css
.item {
  flex-shrink: 1; /* Default: can shrink */
  flex-shrink: 0; /* Won't shrink */
  flex-shrink: 2; /* Shrinks twice as fast */
}
```

### Flex Basis
Sets the initial size before growing/shrinking:

```css
.item {
  flex-basis: 200px;  /* Initial width of 200px */
  flex-basis: 50%;    /* Initial width of 50% */
  flex-basis: auto;   /* Based on content (default) */
}
```

### Flex Shorthand
```css
.item {
  flex: 1;           /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */
  flex: 0 0 200px;   /* flex-grow: 0, flex-shrink: 0, flex-basis: 200px */
  flex: 2 1 auto;    /* flex-grow: 2, flex-shrink: 1, flex-basis: auto */
}
```

## Gap Property

Modern way to add space between flex items:

```css
.container {
  display: flex;
  gap: 1rem;           /* Same gap for rows and columns */
  gap: 1rem 2rem;      /* Different gap for rows and columns */
  row-gap: 1rem;       /* Gap between rows */
  column-gap: 2rem;    /* Gap between columns */
}
```

## Align Self

Override alignment for individual items:

```css
.container {
  display: flex;
  align-items: center; /* All items centered */
  height: 200px;
}

.special-item {
  align-self: flex-start; /* This item aligns to top */
  align-self: flex-end;   /* This item aligns to bottom */
  align-self: stretch;    /* This item stretches */
}
```

```css
  flex-direction: column;
  min-height: 100vh;
```

```css
.media {
  display: flex;

  gap: 1rem;
}
```

```css

  }

  }
```

## Best Practices

1. **Use `flex: 1`** for equal distribution
2. **Use `gap`** instead of margins for spacing
3. **Set `min-width: 0`** on flex items to allow text wrapping
4. **Use `flex-basis`** instead of width for initial sizing
5. **Test on different screen sizes** regularly
6. **Use flex for 1D layouts**, CSS Grid for 2D layouts

## Common Gotchas

1. **Flex items have `min-width: auto`** by default (can cause overflow)
2. **Margins don't collapse** in flex containers
3. **`align-items: baseline`** aligns to text, not container bottom
4. **`justify-content: space-between`** needs at least 2 items

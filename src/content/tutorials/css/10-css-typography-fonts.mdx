# Typography & Web Fonts

Welcome to Typography & Web Fonts! In this tutorial, you'll learn advanced typography techniques, web font loading strategies, and how to create beautiful, readable text layouts that enhance user experience.

## Typography Fundamentals

Typography is more than just choosing fonts. It's about creating hierarchy, improving readability, and enhancing the overall user experience through thoughtful text design.

### Key Typography Concepts

- **Typeface vs Font**: Typeface is the design, font is the file
- **Font Weight**: Thickness of characters (100-900)
- **Font Style**: Normal, italic, oblique
- **Line Height**: Space between lines of text
- **Letter Spacing**: Space between characters
- **Word Spacing**: Space between words

**Understanding Font Loading Strategies**
This demo illustrates three critical font loading approaches:
- **System fonts**: Instant rendering, no network delay, but limited design control
- **font-display: swap**: Shows fallback text immediately, then swaps to custom font when loaded (causes layout shift)
- **font-display: optional**: Only loads custom font on fast connections, prevents layout shift

The loading states simulation shows what users experience during font loading phases.

## Typography Best Practices

### Readability and Accessibility

1. **Sufficient Contrast**: Ensure text meets WCAG guidelines (4.5:1 for normal text)
2. **Readable Font Sizes**: Minimum 16px for body text
3. **Proper Line Height**: 1.4-1.6 for body text
4. **Optimal Line Length**: 45-75 characters per line
5. **Clear Hierarchy**: Use consistent heading scales

### Performance Optimization

```css
/* Font loading optimization */
@font-face {
  font-family: 'CustomFont';
  src: url('font.woff2') format('woff2');
  font-display: swap; /* Show fallback immediately, swap when loaded */
  unicode-range: U+0000-00FF; /* Only load Latin characters */
}

/* Preload critical fonts */
/* <link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin> */

/* System font stack for instant rendering */
font-family: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

**Typography Performance & Accessibility Explained**
These best practices ensure your typography is both beautiful and functional:

**Font Loading Optimization:**
- `font-display: swap` prevents invisible text (FOIT) by showing fallbacks immediately
- `unicode-range` reduces file size by loading only needed character sets
- `preload` hints tell browsers to fetch critical fonts early
- System font stacks provide instant, accessible fallbacks

**Accessibility Guidelines:**
- **4.5:1 contrast ratio** ensures text is readable for users with visual impairments
- **16px minimum** font size accommodates users who need larger text
- **45-75 character line length** optimizes reading speed and comprehension
- **1.4-1.6 line height** provides comfortable spacing for dyslexic users

# CSS Positioning & Z-Index: Control Element Placement

CSS positioning gives you precise control over where elements appear on your page. Combined with z-index, you can create layered interfaces, sticky navigation, tooltips, modals, and much more!

## The Five Types of Positioning

CSS offers five types of positioning, each with unique behaviors:

1. **static** - Default positioning (normal document flow)
2. **relative** - Positioned relative to its normal position
3. **absolute** - Positioned relative to nearest positioned ancestor
4. **fixed** - Positioned relative to the viewport
5. **sticky** - Switches between relative and fixed based on scroll

- Avoid extremely high values (like 999999)

## Common Positioning Problems & Solutions

### Problem 1: Z-Index Not Working
```css
/* ‚ùå Won't work - static positioning */
.element {
  z-index: 100;
}

/* ‚úÖ Works - needs positioning */
.element {
  position: relative;
  z-index: 100;
}
```

### Problem 2: Absolute Element Positioned Wrong
```css
/* ‚ùå Positioned relative to body */
.child {
  position: absolute;
  top: 20px;
}

/* ‚úÖ Parent creates positioning context */
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 20px;
}
```

### Problem 3: Fixed Elements on Mobile
```css
/* ‚ùå Can cause issues on mobile */
.fixed-element {
  position: fixed;
  bottom: 0;
}

/* ‚úÖ Better for mobile */
.fixed-element {
  position: sticky;
  bottom: 0;
}
```

### Problem 4: Sticky Not Working
```css
/* ‚ùå Missing threshold */
.sticky {
  position: sticky;
}

/* ‚úÖ Needs top/bottom/left/right */
.sticky {
  position: sticky;
  top: 0;
}
```

## Positioning Best Practices

### 1. Use Semantic Positioning
```css
/* ‚úÖ Good - clear intent */
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
}

.tooltip {
  position: absolute;
  top: 100%;
  left: 50%;
  transform: translateX(-50%);
}
```

### 2. Plan Your Z-Index Scale
```css
/* ‚úÖ Organized z-index scale */
.dropdown { z-index: 100; }
.sticky-nav { z-index: 200; }
.modal-backdrop { z-index: 1000; }
```

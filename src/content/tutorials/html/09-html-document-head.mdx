# HTML Document Head: SEO and Meta Information

Welcome to the invisible powerhouse of web pages! üîç The HTML `<head>` section contains crucial metadata that affects SEO, social media sharing, performance, and user experience - even though users never see it directly.

## Understanding the Document Head

The `<head>` element contains machine-readable information about the document. It's not displayed on the page but provides essential information to browsers, search engines, and other tools.

<HTMLEditorPreview title="Basic Head Structure" description="Essential elements every head should have:"
initialHtml={`<!DOCTYPE html>
<html lang="en">
<head>
  <!-- Character encoding (must be first) -->
  <meta charset="UTF-8">
  
  <!-- Viewport for responsive design -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <!-- Page title (appears in browser tab) -->
  <title>Your Page Title - Your Site Name</title>
  
  <!-- Meta description for search engines -->
  <meta name="description" content="A compelling description of your page content that appears in search results.">
  
  <!-- Stylesheet links -->
  <link rel="stylesheet" href="styles.css">
  
  <!-- Favicon -->
  <link rel="icon" href="/favicon.ico" type="image/x-icon">
</head>
<body>
  <!-- Visible content goes here -->
</body>
</html>`}
>
</HTMLEditorPreview>

> **‚ö° Performance Tip:** The `<meta charset>` declaration should be within the first 1024 bytes of the document and is usually placed first in the head.

<HTMLEditorPreview title="Dynamic Head Management" description="Managing head content with JavaScript:"
initialHtml={`<!-- Base template with placeholders -->

  <script>
    // Update page meta information
    function updatePageMeta(data) {
      // Update title
      document.title = data.title + ' - Your Site';
      document.getElementById('page-title').textContent = document.title;
      
      // Update description
      const description = document.querySelector('meta[name="description"]');
      description.setAttribute('content', data.description);
      
      // Update Open Graph
      updateMetaProperty('og:title', data.title);
      updateMetaProperty('og:description', data.description);
      updateMetaProperty('og:url', window.location.href);
      
      // Update Twitter Card
      updateMetaName('twitter:title', data.title);
      updateMetaName('twitter:description', data.description);
    }
    
    function updateMetaProperty(property, content) {
      let meta = document.querySelector(\`meta[property="\${property}"]\`);
      if (!meta) {
        meta = document.createElement('meta');
        meta.setAttribute('property', property);
        document.head.appendChild(meta);
      }
      meta.setAttribute('content', content);
    }
    
    function updateMetaName(name, content) {
      let meta = document.querySelector(\`meta[name="\${name}"]\`);
      if (!meta) {
        meta = document.createElement('meta');
        meta.setAttribute('name', name);
        document.head.appendChild(meta);
      }
      meta.setAttribute('content', content);
    }
    
    // Example usage for SPA route changes
    function handleRouteChange(route) {
      const pageData = {

          title: 'About Us',
          description: 'Learn about our company and mission.'
        },
        '/contact': {
          title: 'Contact Us',
          description: 'Get in touch with our team.'
        }
      };
      
      if (pageData[route]) {
        updatePageMeta(pageData[route]);
      }
    }
  </script>
</head>`}
